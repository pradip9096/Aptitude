Symbol Chain

\(\mathbb{N} \subset \mathbb{W} \subset \mathbb{Z} \subset \mathbb{Q} \subset \mathbb{R}\)

‚ö° Number Classification Exam Cheat Sheet

1. The Snapshot Hierarchy

  • Concept: Numbers are nested like boxes. If a number belongs to a smaller box, it automatically belongs to all bigger boxes around it.
  • The Symbol Chain: \(\mathbb{N} \subset \mathbb{W} \subset \mathbb{Z} \subset \mathbb{Q} \subset \mathbb{R}\)
    • \(\mathbb{N}\) (Natural): \(\{1, 2, 3 \dots\}\)
    • \(\mathbb{W}\) (Whole): \(\{0, 1, 2 \dots\}\)
    • \(\mathbb{Z}\) (Integer): \(\{\dots -1, 0, 1 \dots\}\)
    • \(\mathbb{Q}\) (Rational): Fractions, terminating or repeating decimals.
    • \(\mathbb{I}\) (Irrational): Non-terminating, non-repeating decimals (\(\pi, \sqrt{2}\)).
    • \(\mathbb{R}\) (Real): All of the above.

2. Speed-Identification Shortcuts

Rational vs. Irrational (The "Pattern Test"):

  • Rational (\(\mathbb{Q}\)): Look for a End or a Pattern.
    • \(0.5\) (Ends) \(\rightarrow\) Rational.
    • \(0.333\dots\) (Pattern) \(\rightarrow\) Rational.
    • \(\sqrt{16}\) (Perfect Square) \(\rightarrow\) Rational (4).
  • Irrational (\(\mathbb{I}\)): Look for Chaos.
    • \(0.101101110\dots\) (No repeating pattern) \(\rightarrow\) Irrational.
    • \(\pi, e\) \(\rightarrow\) Irrational.
    • \(\sqrt{2}, \sqrt{3}, \sqrt{5}\) (Non-perfect roots) \(\rightarrow\) Irrational.

Prime Number Check (The "6n Rule"):

  • Rule: All primes greater than 3 can be written in the form \(6n - 1\) or \(6n + 1\).
  • Usage: If a number cannot be written this way, it is definitely not prime.
  • Warning: If it can be written this way, it might be prime (but check further).
Example: Is 35 prime? \(35 = 6(6) - 1\). Fits the rule, but \(35 = 5 \times 7\). So, not prime. (Use as a quick filter, not a guarantee).

Co-Prime Identification:

  • Two numbers are Co-Prime if their HCF is 1.
  • Shortcut: Any two consecutive integers are always co-prime (e.g., 20 and 21).

3. MCQ Elimination Tactics (True/False Logic)

Strategy: Use the "Inner Circle" logic.

  • Statement: "All [Small Set] are [Big Set]." \(\rightarrow\) TRUE.
    • Example: "All Integers are Rational." (True, \(\mathbb{Z}\) is inside \(\mathbb{Q}\)).
  • Statement: "All [Big Set] are [Small Set]." \(\rightarrow\) FALSE.
    • Example: "All Real numbers are Integers." (False, \(\mathbb{R}\) is outside \(\mathbb{Z}\)).

Logical Traps to Exploit:

  • If an option says "Positive Numbers," verify if it excludes 0.
  • If an option says "Non-Negative Numbers," it includes 0.

4. Essential Properties Table

Type Key Property Examples
Prime Exactly 2 factors (1 and itself). \(2, 3, 5, 7, 11, 13, 17, 19\)
Composite More than 2 factors. \(4, 6, 8, 9, 10, 12\)
Even Divisible by 2 (\(2k\)). \(\dots -4, -2, 0, 2, 4 \dots\)
Odd Not divisible by 2 (\(2k+1\)). \(\dots -3, -1, 1, 3, 5 \dots\)
Perfect Sum of factors (excluding itself) = Number. \(6\) (\(1+2+3\)), \(28\) (\(1+2+4+7+14\))

5. Word Problem "Trigger Words"

  • "Distinct Prime Factors": Count types of primes, not total primes (e.g., \(12 = 2 \times 2 \times 3\) has 2 distinct primes: 2 and 3).
  • "Consecutive Integers": Write as \(x, x+1, x+2\).
  • "Non-negative Integers": Remember to start checking from 0, not 1.
  • "Reciprocal": The flip of the number (\(x \rightarrow 1/x\)). Zero has no reciprocal.

6. The "Exam Trap" Alert ‚ö†Ô∏è

Trap 1: "1 is a Prime Number."

Verdict: FALSE. 1 is a Unit (neither prime nor composite). The smallest prime is 2.

Trap 2: "0 is neither Positive nor Negative."

Verdict: TRUE. Zero is neutral.

Trap 3: "0 is an Odd number."

Verdict: FALSE. 0 is divisible by 2 (\(0 \div 2 = 0\)). Therefore, 0 is Even.

Trap 4: \(\sqrt{x^2} = \pm x\).

Verdict: FALSE in arithmetic contexts. The radical symbol \(\sqrt{\dots}\) implies the principal (positive) root only. \(\sqrt{(-3)^2} = \sqrt{9} = 3\).
