Quick Reference

\(n^2 = n \times n\)

No squares end in 2, 3, 7, or 8

‚ö° Square Exam Cheat Sheet

1. The Snapshot Definition

  • Concept: Multiplying a number by itself (\(n \times n\)).
  • The Sign Rule: The square of any real number is always Positive (\((-5)^2 = 25\)).
  • The "Impossible Endings" Rule: A perfect square never ends in the digits 2, 3, 7, or 8.
    • Action: If an MCQ option ends in these digits, eliminate it immediately.
  • The Digital Root Rule: The digital root (sum of digits reduced to a single digit) of a perfect square is always 1, 4, 7, or 9.

2. Speed-Math Shortcuts (Mental Math)

Numbers Ending in 5:

  • Technique: For a number \(n5\), multiply the first digit \(n\) by the next integer \((n+1)\). Append \(25\) to the end.
Example: \(65^2\)
Step 1: \(6 \times 7 = 42\)
Step 2: Append \(25 \rightarrow \mathbf{4225}\)

Base 50 Method (For numbers 41‚Äì59):

  • Technique: Compare the number to 50. Add/Subtract the difference from 25. Append the square of the difference.
Example: \(54^2\) (\(+4\) from 50)
Part 1: \(25 + 4 = 29\)
Part 2: \(4^2 = 16\)
Result: 2916
Example: \(47^2\) (\(-3\) from 50)
Part 1: \(25 - 3 = 22\)
Part 2: \(3^2 = 09\) (Must be two digits)
Result: 2209

Base 100 Method (For numbers 91‚Äì109):

  • Technique: Add/Subtract the difference directly to/from the number itself. Append the square of the difference.
Example: \(106^2\) (\(+6\) from 100)
Part 1: \(106 + 6 = 112\)
Part 2: \(6^2 = 36\)
Result: 11236
Example: \(98^2\) (\(-2\) from 100)
Part 1: \(98 - 2 = 96\)
Part 2: \(2^2 = 04\)
Result: 9604

3. MCQ Elimination Tactics (Verification)

Strategy: Don't calculate the full square if you can spot the imposter.

1. Last Digit Logic:

The last digit of the square depends only on the last digit of the base.

  • \(1 \text{ or } 9 \rightarrow \text{Ends in } 1\)
  • \(2 \text{ or } 8 \rightarrow \text{Ends in } 4\)
  • \(3 \text{ or } 7 \rightarrow \text{Ends in } 9\)
  • \(4 \text{ or } 6 \rightarrow \text{Ends in } 6\)
  • \(5 \rightarrow \text{Ends in } 5\)
  • \(0 \rightarrow \text{Ends in } 0\)

2. Range Estimation:

Round to the nearest 10 to find the minimum and maximum possible answer.

Task: \(58^2\)
Estimate: \(60^2 = 3600\). The answer must be slightly less than 3600. Eliminate options like 2500 or 4200.

4. Essential Formulas

Concept Formula
Sum Square \((a+b)^2 = a^2 + 2ab + b^2\)
Difference Square \((a-b)^2 = a^2 - 2ab + b^2\)
Difference of Squares \(a^2 - b^2 = (a+b)(a-b)\)
Sum of Squares \(a^2 + b^2 = (a+b)^2 - 2ab\)
Square of 3 terms \((a+b+c)^2 = a^2 + b^2 + c^2 + 2(ab+bc+ca)\)

5. Word Problem "Trigger Words"

If you see these phrases, you likely need to square a number or solve a quadratic:

  • "Find the Area of a square..."
  • "Calculated by multiplying the value by itself..."
  • "Depends on the square of the velocity..." (Physics contexts)
  • "Arrange soldiers in a solid square formation..."

6. The "Exam Trap" Alert ‚ö†Ô∏è

Trap 1: \(-x^2\) vs \((-x)^2\)
  • \(-6^2 = -36\) (Negative)
  • \((-6)^2 = +36\) (Positive)
  • Fix: Watch the brackets carefully.
Trap 2: Decimal Doubling
  • Problem: \(0.3^2 = ?\)
  • Trap: Thinking it is \(0.9\).
  • Fix: Squaring doubles the number of decimal places. 1 place becomes 2 places. Answer is 0.09.
Trap 3: \(a^2 + b^2 \ne (a+b)^2\)
  • Example: \(3^2 + 4^2 = 9 + 16 = 25\)
  • Error: \((3+4)^2 = 7^2 = 49\). They are not the same. Do not distribute the exponent.
